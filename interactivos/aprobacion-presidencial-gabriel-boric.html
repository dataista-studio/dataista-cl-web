Los datos de las encuestas fueron sacados de las páginas web de cada una de las empresas, las cuales han sido compiladas por Wikipedia para su fácil acceso.

Métodos

Para el cálculo de la línea de tendencia se usa un promedio ponderado por el número de respuestas en cada encuesta. Además, se usa una ventana variable de 1, 3, ó 5 encuestas centradas en el correspondiente dato. Es decir, si se ocupa una ventana de 3 encuestas, se usa el punto inmediatamente anterior, el punto actual, y el punto inmediatamente posterior.

Los porcentajes absolutos se refieren a los valores considerando los "No sabe/no responde", tal cual aparecen en las publicaciones de las encuestas. En el caso de los valores relativos, éstos se calculan sólo considerando las opciones "Aprueba" y "Desaprueba", por ende ambos porcentajes siempre suman 100%.
